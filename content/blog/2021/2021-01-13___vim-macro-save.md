1 度作ったマクロを保存して、 vim を開き直したときも使いまわしたいときの方法。
マクロ `@b` に保存する例で説明します。

## .vimrc に保存する

### q で macro を記録する

`qb` → 操作 → `q` で記録。

### `.vimrc` に追記する

```
let @b = 'Gk$F_lvEygg$p
```

#### (外部クリップボードにコピーしたりしたいとき)

```
"bpyy
```

`"b` b を選択 →`p` 貼り付け → `yy` コピーで外部クリップボードにコピーしておきます。

```
:echo @b
:reg b
```

で確認もできます。
